# 基于 AI 辅助的视频帧级打标工具设计与实现——面向 LoRA 模型训练的数据标注系统

**摘要**

随着开源 AI 视频生成模型的快速发展，LoRA（Low-Rank Adaptation）微调技术已成为优化视频模型的主流方式。然而，高质量的精细化视频标注数据制作困难、人工打标效率低下等问题严重制约了 LoRA 训练的成效。本文设计并实现了一款专为 LoRA 训练场景服务的 AI 辅助视频打标工具，融合计算机视觉、前端交互设计与 AI 视觉识别技术，实现了帧级精细预览、AI 连续帧画面描述自动生成、标签与片段导出等功能。实验表明，该工具可显著提升视频标注效率，为开源 AI 视频模型的 LoRA 微调提供高质量标注数据支撑。

**关键词**: 计算机视觉；AI 辅助标注；视频打标;LoRA 模型训练;帧级数据处理;视频剪辑技术

## 第一章 绪论

### 1.1 研究背景

近年来，以 Stable Video Diffusion、Wanx、ModelScope 为代表的开源 AI 视频生成模型取得了突破性进展。LoRA（Low-Rank Adaptation）微调技术因其参数高效、训练成本低等优势，已成为开源社区优化视频模型的主流方式。然而，LoRA 训练的核心在于高质量、精细化的视频标注数据——标注质量直接决定模型的生成效果与主题一致性。

当前视频打标面临以下痛点：
1. **人工效率低下**: 逐帧浏览标注耗时费力，一个 5 分钟的视频可能需要数小时完成标注
2. **帧级微调困难**: 通用视频剪辑软件缺乏精准的帧定位与片段选择功能
3. **无 AI 辅助**: 无法自动识别画面内容，依赖人工描述，主观性强且标准不一
4. **格式不匹配**: 现有工具无法直接输出适配 LoRA 训练的标签导出格式

因此，设计一款面向 LoRA 训练场景的 AI 辅助视频打标工具具有重要的研发价值。

### 1.2 研究意义

**实践意义**: 
- 为开源 AI 视频模型 LoRA 训练提供高效的标注工具

### 2.1 视频帧处理技术

#### 2.1.1 视频编码基础

视频由连续帧组成，常见编码格式包括 H.264、H.265（HEVC）。解码过程涉及：
- **解封装**: 分离视频流与音频流
- **熵解码**: 逆霍夫曼编码、算术解码
- **反量化与反 DCT**: 恢复像素值

### 2.3 Base64 图像编码

为将图像传递给 AI 模型，需转换为 Base64 Data URL 格式：

```python
import base64, io
from PIL import Image

def encode_image_to_data_url(image):
    """PIL 图像转 Base64 Data URL"""
    buffered = io.BytesIO()
    image.save(buffered, format="PNG")
    encoded = base64.b64encode(buffered.getvalue()).decode("utf-8")
    return f"data:image/png;base64,{encoded}"
```

### 4.1 视频解析与预览模块

#### 4.1.1 设计思路

基于 OpenCV 实现视频解封装，提取帧数据并缓存至内存。采用均匀采样策略降低资源消耗：
- **目标 FPS**: 16 帧/秒（可配置）
- **分辨率缩放**: 保持宽高比缩放到 720p 高度
- **进度反馈**: 加载窗口显示实时处理进度

#### 4.1.2 实现过程

```python
def preprocess_frames(self):
    """视频帧预处理"""
    target_fps = self.config.getint('PROCESSING', 'target_frame_rate', fallback=16)
    
    # 计算采样间隔
    if self.fps > target_fps:
        frame_interval = self.fps / target_fps
    else:
        frame_interval = 1
    
    # 均匀采样并缓存
    processed_frames = []
    for i in range(total_frames):
        if i % int(frame_interval) == 0:
            ret, frame = self.cap.read()
            if ret:
                frame_rgb = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
                frame_resized = self.resize_to_720p(frame_rgb)
                processed_frames.append(frame_resized)
    
    # 更新元数据
    self.total_frames = len(processed_frames)
    self.fps = target_fps if self.fps > target_fps else self.fps
```

def set_start_frame(self):
    """设置开始帧"""
    self.start_frame = self.current_frame
    
    # 自动计算结束帧（5 秒时长）
    segment_duration = self.config.getint('PROCESSING', 'segment_duration', fallback=5)
    frames_5_seconds = int(segment_duration * self.fps)
    self.end_frame = min(self.current_frame + frames_5_seconds, self.total_frames - 1)
    
    self.draw_tag_markers()

def add_tag(self):
    """添加标签"""
    tag_text = self.tag_entry.get("1.0", tk.END).strip()
    
    if self.start_frame >= self.end_frame:

    tag_info = {
        "start": self.start_frame,
        "end": self.end_frame,
        "tag": tag_text

### 4.3 AI 连续帧识别与标签生成模块

#### 4.3.1 设计思路

这是项目的核心创新点：
1. **帧序列提取**: 从选中区间均匀采样最多 15 帧
2. **图像编码**: PIL 缩放→PNG→Base64 Data URL
3. **API 调用**: 构造多模态消息体发送给 Qwen-VL
4. **结果验证**: 
   - 移除模型思考标签（``）
   - 敏感词检测与重试机制（最多 10 次）
   - 长度检查（≥50 字符）

#### 4.3.2 OllamaAPIClient 类设计

```python
class OllamaAPIClient:
    """Ollama API 客户端封装"""
    
    def __init__(self, config):
        self.client = OpenAI(
            api_key=config.get('OLLAMA', 'api_key'),
            base_url=config.get('OLLAMA', 'api_base_url')
        )
    
    def extract_frames(self, processed_frames, start_frame, end_frame):
        """提取采样帧"""
        max_sample_frames = self.config.getint('PROCESSING', 'max_sample_frames', fallback=15)
        
        total_frames = end_frame - start_frame + 1
        sample_count = min(max_sample_frames, total_frames)
        
        if total_frames <= sample_count:
            indices = list(range(start_frame, end_frame + 1))
        else:
            indices = np.linspace(start_frame, end_frame, sample_count, dtype=int)
        
        frames = []
        for i in indices:
            if i < len(processed_frames):
                pil_frame = Image.fromarray(processed_frames[i])
                frames.append(pil_frame)
        
        return frames
    
    def generate_caption(self, frames, prompt_text=None):
        """生成标签描述"""
        image_urls = [self.convert_to_base64(frame) for frame in frames]
        
        messages = [{
            "role": "user",
            "content": [
                {"type": "text", "text": prompt_text or DEFAULT_PROMPT},
                * [{"type": "image_url", "image_url": {"url": url}} for url in image_urls]
            ]
        }]
        
        response = self.client.chat.completions.create(
            model="qwen3-vl:9b",
            messages=messages,
            max_tokens=16384,
            temperature=0.3
        )
        
        return response.choices[0].message.content
```

#### 4.3.3 敏感词过滤机制

```python
def generate_caption_with_filters(self, frames, prompt):
    """带过滤的标签生成"""
    filter_words = self.get_filter_words()
    
    for attempt in range(10):
        caption = self.generate_caption(frames, prompt)
        
        # 移除思考标签
        caption = re.sub(r"", "", caption, flags=re.DOTALL).strip()
        
        # 检查敏感词
        if any(word in caption.lower() for word in filter_words):
            logger.info(f"发现敏感词，重新生成 (尝试{attempt+1}/10)")
            continue
        
        # 长度验证
        if len(caption) >= 50:
            return caption
    
    # 强制移除敏感词后返回
    for word in filter_words:
        caption = caption.replace(word, '')
    return caption.strip() or "视频描述内容已被过滤"
```

def save_and_replace_video(self):
    """保存起始帧到末尾帧区间为新视频，替换原视频"""
    if self.start_frame >= self.end_frame:
        messagebox.showwarning("警告", "请先设置有效的起止帧")
        return
    
    # 获取导出参数
    output_fps = float(self.export_fps.get()) or self.fps
    
    # 创建临时文件
    temp_path = os.path.join(video_dir, f"_temp_{video_name}")
    
    # 打开视频写入器
    fourcc = cv2.VideoWriter_fourcc(*'mp4v')
    out = cv2.VideoWriter(temp_path, fourcc, output_fps, (width, height))
    
    # 跳转至起始帧并写入
    self.cap.set(cv2.CAP_PROP_POS_FRAMES, self.start_frame)
    frames_written = 0
    
    while True:
        ret, frame = self.cap.read()
        if not ret or self.cap.get(cv2.CAP_PROP_POS_FRAMES) > self.end_frame:
            break
        
        out.write(frame)
        frames_written += 1
    
    out.release()
    
    # 替换原文件
    os.remove(self.video_path)
    os.rename(temp_path, self.video_path)
```

def export_tags(self):
    """导出标记片段"""
    for i, tag in enumerate(self.tags):
        # 生成安全文件名
        safe_tag = "".join(c for c in tag["tag"] if c.isalnum() or c in (' ','-','_'))
        safe_tag = safe_tag[:10] if len(safe_tag) > 10 else safe_tag
        safe_tag = safe_tag.replace(" ", "_")
        
        filename = f"video_{i+1:03d}_{safe_tag}"
        video_path = os.path.join(self.export_dir, f"{filename}.mp4")
        txt_path = os.path.join(self.export_dir, f"{filename}.txt")
        
        # 视频写入
        out = cv2.VideoWriter(video_path, fourcc, export_fps, (width, height))
        for frame_idx in range(tag["start"], min(tag["end"] + 1, len(self.processed_frames))):
            frame_bgr = cv2.cvtColor(self.processed_frames[frame_idx], cv2.COLOR_RGB2BGR)
            out.write(frame_bgr)
        out.release()
        
        # 标签写入
        with open(txt_path, 'w', encoding='utf-8') as f:
            f.write(tag["tag"])
```
