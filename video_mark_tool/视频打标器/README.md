# 视频打标器 (Video Tagger)

一个基于 Python 和 tkinter 开发的视频标注工具，支持手动标记和 AI 自动生成标签功能。

## 功能特性

### 核心功能
- 📹 **视频加载与播放** - 支持常见视频格式加载，可播放/暂停、逐帧导航
- 🏷️ **帧标记** - 可设置开始帧和结束帧，标记视频片段
- 📝 **标签管理** - 添加、编辑、删除标签，支持多行文本输入
- 💾 **记录保存** - 支持保存和加载标记记录（JSON 格式）
- 📤 **片段导出** - 导出所有标记的视频片段，可设置导出帧率

### AI 功能
- 🤖 **AI 生成标签预设** - 使用 Qwen 视觉语言模型自动生成标签建议
- 🔄 **自动分段识别** - AI 自动分析视频并生成分段标签
- ⚡ **批量重新生成** - 支持重新生成所有标签并导出

### UI 特性
- 🎨 **可调节界面** - 支持窗口大小调整、字体大小缩放
- 📋 **标签预设** - 支持手动添加和 AI 生成的标签预设
- 🖱️ **右键菜单** - 标签列表和预设列表支持右键快捷操作
- 🎯 **进度条导航** - 可视化进度条快速定位视频帧

## 系统要求

- Python 3.8+
- Windows 10/11 推荐
- 建议使用 NVIDIA GPU（用于 AI 功能）
